# 程度和数量副词

|                  | 表示程度的副词          | 表示数量的副词        |
| :--------------- | :---------------------- | :-------------------- |
| 和肯定形一起使用 | `よく` わかります       | `たくさん` あります   |
|                  | `だいたい` わかります   | `すこし` あります     |
|                  | `すこし` わかります     |                       |
| 和否定形一起使用 | `あまり` わかりません   | `あまり` ありません   |
|                  | `ぜんぜん` わかりません | `ぜんぜん` ありません |
